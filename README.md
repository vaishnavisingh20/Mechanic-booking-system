#  Mechanic Booking System

A full-stack mechanic service booking platform built with **Next.js 15, TypeScript, MongoDB, JWT Authentication, and Tailwind CSS**.  
The system provides secure role-based access for **Admin, Mechanic, and Guest users** to manage mechanics, users, and service bookings.

##  Features

- JWT Authentication with HTTP-only cookies
- Role-Based Access Control (Admin, Mechanic, Guest)
- Admin dashboard for user and booking management
- Mechanic dashboard for handling service requests
- Guest registration and booking creation
- Booking management system
- Protected routes and API endpoints
- Responsive UI with Tailwind CSS
